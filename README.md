# Reachly Frontend

This is a simple Next.js application for Reachly.

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Built with Next.js 14
- TypeScript support
- Tailwind CSS for styling
- Responsive design
- Your Reachly logos included
